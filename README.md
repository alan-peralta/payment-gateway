# Teste

Projeto de um gateway de pagamento. Nela é possível depositar e realizar transferências de dinheiro entre usuários. Temos 2 tipos de usuários, os comuns e lojistas, ambos têm carteira com dinheiro e realizam transferências entre eles.

## Iniciando o projeto
Usei SAIL para conteinerizar o projeto.

**Iniciar o projeto**

    ./vendor/bin/sail up -d

**Rodar migrations**

    ./vendor/bin/sail migrate
